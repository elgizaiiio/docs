## Generate Images

### Basic Generation

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "megsy-images",
        "prompt": "A serene mountain landscape at sunset with a lake reflection",
        "size": "1024x1024",
        "quality": "hd"
    }
)

image_url = response.json()["data"][0]["url"]
print(f"Image URL: {image_url}")
```

## Image Enhancement Tools

### Clarity Upscaler

Enhance image resolution and quality:

```python
response = requests.post(
    "https://api.megsyai.com/v1/images/upscale",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("low_res.jpg", "rb")},
    data={
        "tool": "clarity-upscaler",
        "scale": 4  # 2x or 4x upscaling
    }
)

upscaled_url = response.json()["url"]
```

## Image Templates

Use ready-made templates for quick creation:

### How Templates Work

1. Choose a template style
2. Upload your photo
3. AI generates a similar image with your photo


## Workflow Examples

### Complete Image Workflow

```python
# 1. Generate base image
generate_response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "megsy-images",
        "prompt": "Modern office interior"
    }
)

image_url = generate_response.json()["data"][0]["url"]

# 2. Edit the image
edit_response = requests.post(
    "https://api.megsyai.com/v1/images/edit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": requests.get(image_url).content},
    data={
        "model": "qwen-edit",
        "prompt": "Add plants and natural lighting"
    }
)

# 3. Upscale for final quality
upscale_response = requests.post(
    "https://api.megsyai.com/v1/images/upscale",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": requests.get(edit_response.json()["url"]).content},
    data={"tool": "clarity-upscaler", "scale": 4}
)

final_image = upscale_response.json()["url"]
```
