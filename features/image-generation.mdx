## Features

### Text-to-Image
Create images from simple text descriptions:

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "dall-e-3",
        "prompt": "A cute cat wearing sunglasses on a beach",
        "size": "1024x1024",
        "quality": "hd",
        "n": 1
    }
)

image_url = response.json()["data"][0]["url"]
print(f"Image URL: {image_url}")
```

### Image Editing
Edit existing images by adding or removing elements:

```python
response = requests.post(
    "https://api.megsyai.com/v1/images/edit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={
        "image": open("original.png", "rb"),
        "mask": open("mask.png", "rb")
    },
    data={
        "prompt": "Add a clear blue sky",
        "size": "1024x1024"
    }
)
```

### Image Variations
Create diverse variations of an existing image:

```python
response = requests.post(
    "https://api.megsyai.com/v1/images/variations",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("original.png", "rb")},
    data={
        "n": 3,
        "size": "1024x1024"
    }
)
```
