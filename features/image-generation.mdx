## الميزات

### توليد من النص
أنشئ صور من وصف نصي بسيط:

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "dall-e-3",
        "prompt": "قطة لطيفة ترتدي نظارة شمسية على شاطئ البحر",
        "size": "1024x1024",
        "quality": "hd",
        "n": 1
    }
)

image_url = response.json()["data"][0]["url"]
print(f"Image URL: {image_url}")
```

### تعديل الصور
عدّل الصور الموجودة بإضافة أو إزالة عناصر:

```python
response = requests.post(
    "https://api.megsyai.com/v1/images/edit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={
        "image": open("original.png", "rb"),
        "mask": open("mask.png", "rb")
    },
    data={
        "prompt": "أضف سماء زرقاء صافية",
        "size": "1024x1024"
    }
)
```

### تنويعات الصور
أنشئ نسخ متنوعة من صورة موجودة:

```python
response = requests.post(
    "https://api.megsyai.com/v1/images/variations",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("original.png", "rb")},
    data={
        "n": 3,
        "size": "1024x1024"
    }
)
```

## الأحجام المتاحة

يدعم Megsy AI عدة أحجام للصور:

| الحجم | الأبعاد | الاستخدام المثالي |
|-------|---------|-------------------|
| صغير | 512x512 | معاينة سريعة |
| متوسط | 1024x1024 | استخدام عام |
| كبير | 1792x1024 | صور أفقية |
| عمودي | 1024x1792 | صور عمودية |

## جودة الصور

### Standard
جودة قياسية مناسبة لمعظم الاستخدامات:
- سرعة توليد أعلى
- تكلفة أقل
- جودة جيدة

### HD
جودة عالية للاستخدامات الاحترافية:
- تفاصيل أدق
- ألوان أفضل
- دقة أعلى

## الأنماط الفنية

يمكنك تحديد النمط الفني للصورة:

```python
styles = [
    "realistic",      # واقعي
    "artistic",       # فني
    "anime",          # أنمي
    "3d-render",      # ثلاثي الأبعاد
    "oil-painting",   # رسم زيتي
    "watercolor",     # ألوان مائية
    "sketch",         # رسم تخطيطي
    "digital-art"     # فن رقمي
]

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "prompt": "منظر طبيعي جميل",
        "style": "oil-painting"
    }
)
```
