## Key Features

### Intelligent Conversation
```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/chat/completions",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "gpt-5",
        "messages": [
            {"role": "system", "content": "You are a helpful and intelligent assistant"},
            {"role": "user", "content": "What's the best way to learn programming?"}
        ],
        "temperature": 0.7,
        "max_tokens": 1000
    }
)

print(response.json()["choices"][0]["message"]["content"])
```

### Multi-turn Conversation
```python
messages = [
    {"role": "system", "content": "You are a programming expert"},
    {"role": "user", "content": "How do I learn Python?"},
    {"role": "assistant", "content": "Start with the basics..."},
    {"role": "user", "content": "What's next?"}
]

response = requests.post(
    "https://api.megsyai.com/v1/chat/completions",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={"model": "gpt-5", "messages": messages}
)
```

### Streaming
```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/chat/completions",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "gpt-5",
        "messages": [{"role": "user", "content": "Write a short story"}],
        "stream": True
    },
    stream=True
)

for line in response.iter_lines():
    if line:
        print(line.decode('utf-8'))
```

## Control Parameters

### Temperature
Controls response randomness:
```python
# Precise and deterministic
temperature = 0.2

# Balanced
temperature = 0.7

# Creative
temperature = 1.0
```

### Max Tokens
Maximum response length:
```python
max_tokens = 500   # Short response
max_tokens = 1000  # Medium response
max_tokens = 4000  # Long response
```

### Top P
Controls response diversity:
```python
top_p = 0.1  # Focused responses
top_p = 0.9  # Diverse responses
```

## Assistants

### Create Assistant
```python
response = requests.post(
    "https://api.megsyai.com/v1/assistants",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "name": "Programming Assistant",
        "instructions": "You are an expert programming assistant",
        "model": "gpt-5",
        "tools": [{"type": "code_interpreter"}]
    }
)

assistant_id = response.json()["id"]
```

### Create Thread
```python
thread_response = requests.post(
    "https://api.megsyai.com/v1/threads",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)

thread_id = thread_response.json()["id"]
```

### Send Message
