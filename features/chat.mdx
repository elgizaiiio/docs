## Key Features

### Intelligent Conversation

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/chat/completions",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "megsy",
        "messages": [
            {"role": "user", "content": "What's the best way to learn programming?"}
        ],
        "temperature": 0.7,
        "max_tokens": 1000
    }
)

print(response.json()["choices"][0]["message"]["content"])
```

### Web Search Integration

Search the web directly from your conversation:

```python
response = requests.post(
    "https://api.megsyai.com/v1/chat/completions",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "megsy",
        "messages": [
            {"role": "user", "content": "What are today's top tech news?"}
        ],
        "tools": [{"type": "web_search"}]
    }
)
```

**Example queries:**
- "What's the weather in Cairo today?"
- "Latest news about AI technology"
- "Find information about Python 3.12 features"
- "Search for the best restaurants near me"

## Multi-turn Conversations

Megsy maintains context across multiple messages:

```python
messages = [
    {"role": "user", "content": "What's the capital of France?"},
    {"role": "assistant", "content": "The capital of France is Paris."},
    {"role": "user", "content": "What's the population?"},
    {"role": "assistant", "content": "Paris has a population of approximately 2.2 million people in the city proper."},
    {"role": "user", "content": "What are the top attractions there?"}
]

response = requests.post(
    "https://api.megsyai.com/v1/chat/completions",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={"model": "megsy", "messages": messages}
)
```

## Streaming Responses

Get responses in real-time as they're generated:

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/chat/completions",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "megsy",
        "messages": [{"role": "user", "content": "Write a short story"}],
        "stream": True
    },
    stream=True
)

for line in response.iter_lines():
    if line:
        print(line.decode('utf-8'))
```

## Control Parameters

### Temperature
Controls response creativity:

```python
# Precise and focused (0.0 - 0.3)
temperature = 0.2

# Balanced (0.4 - 0.7)
temperature = 0.7

# Creative and diverse (0.8 - 1.0)
temperature = 0.9
```

### Max Tokens
Maximum response length:

```python
max_tokens = 500   # Short response
max_tokens = 1000  # Medium response
max_tokens = 2000  # Long response
```

### Top P
Controls response diversity:

```python
top_p = 0.1  # Focused responses
top_p = 0.9  # Diverse responses
```
