## الميزات الرئيسية

### محادثة ذكية
```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/chat/completions",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "gpt-5",
        "messages": [
            {"role": "system", "content": "أنت مساعد ذكي ومفيد"},
            {"role": "user", "content": "ما هي أفضل طريقة لتعلم البرمجة؟"}
        ],
        "temperature": 0.7,
        "max_tokens": 1000
    }
)

print(response.json()["choices"][0]["message"]["content"])
```

### محادثة متعددة الأدوار
```python
messages = [
    {"role": "system", "content": "أنت خبير في البرمجة"},
    {"role": "user", "content": "كيف أتعلم Python؟"},
    {"role": "assistant", "content": "ابدأ بتعلم الأساسيات..."},
    {"role": "user", "content": "وماذا بعد ذلك؟"}
]

response = requests.post(
    "https://api.megsyai.com/v1/chat/completions",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={"model": "gpt-5", "messages": messages}
)
```

### البث المباشر (Streaming)
```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/chat/completions",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "gpt-5",
        "messages": [{"role": "user", "content": "اكتب قصة قصيرة"}],
        "stream": True
    },
    stream=True
)

for line in response.iter_lines():
    if line:
        print(line.decode('utf-8'))
```

## معاملات التحكم

### Temperature
يتحكم في عشوائية الإجابات:
```python
# إجابات دقيقة ومحددة
temperature = 0.2

# إجابات متوازنة
temperature = 0.7

# إجابات إبداعية
temperature = 1.0
```

### Max Tokens
الحد الأقصى لطول الإجابة:
```python
max_tokens = 500   # إجابة قصيرة
max_tokens = 1000  # إجابة متوسطة
max_tokens = 4000  # إجابة طويلة
```

### Top P
التحكم في تنوع الإجابات:
```python
top_p = 0.1  # إجابات محددة
top_p = 0.9  # إجابات متنوعة
```

## المساعدات الذكية (Assistants)

### إنشاء مساعد
```python
response = requests.post(
    "https://api.megsyai.com/v1/assistants",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "name": "مساعد البرمجة",
        "instructions": "أنت مساعد برمجي خبير",
        "model": "gpt-5",
        "tools": [{"type": "code_interpreter"}]
    }
)

assistant_id = response.json()["id"]
```

### إنشاء محادثة
```python
thread_response = requests.post(
    "https://api.megsyai.com/v1/threads",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)

thread_id = thread_response.json()["id"]
```

### إرسال رسالة
```python
message_response = requests.post(
```
