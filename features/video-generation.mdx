## Features

### Text-to-Video
Create videos from text descriptions:

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/videos/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "runway-gen3",
        "prompt": "Ocean waves crashing on the beach at sunset, slow motion",
        "duration": 5,
        "resolution": "1080p",
        "fps": 24
    }
)

video_url = response.json()["data"]["url"]
print(f"Video URL: {video_url}")
```

### Image Animation
Convert a static image into an animated video:

```python
response = requests.post(
    "https://api.megsyai.com/v1/videos/animate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("image.png", "rb")},
    data={
        "motion_prompt": "Camera slowly moves from left to right",
        "duration": 3
    }
)
```

### Video Editing
Edit existing videos with effects:

```python
response = requests.post(
    "https://api.megsyai.com/v1/videos/edit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"video": open("video.mp4", "rb")},
    data={
        "edit_prompt": "Add slow motion effect",
        "start_time": 2,
        "end_time": 5
    }
)
```

## Advanced Processing

### Add Text
```python
response = requests.post(
    "https://api.megsyai.com/v1/videos/add-text",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"video": open("video.mp4", "rb")},
    data={
        "text": "Megsy AI",
        "position": "center",
        "font": "Arial",
        "size": 48,
        "color": "#FFFFFF",
        "animation": "fade-in"
    }
)
```

### Add Music
```python
response = requests.post(
    "https://api.megsyai.com/v1/videos/add-audio",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={
        "video": open("video.mp4", "rb"),
        "audio": open("music.mp3", "rb")
    },
    data={
        "volume": 0.7,
        "fade_in": 1,
        "fade_out": 1
    }
)
```

### Merge Videos
```python
response = requests.post(
    "https://api.megsyai.com/v1/videos/merge",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={
        "video1": open("video1.mp4", "rb"),
        "video2": open("video2.mp4", "rb")
    },
    data={
        "transition": "fade",
        "transition_duration": 1
    }
)
```
