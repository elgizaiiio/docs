## Generate Videos

### Basic Video Generation

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/videos/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "megsy-v1",
        "prompt": "A serene ocean sunset with waves gently crashing on the beach",
        "duration": 5,
        "resolution": "1080p",
        "include_audio": True
    }
)

video_url = response.json()["data"]["url"]
print(f"Video URL: {video_url}")
```

## Camera Movements

Specify camera motion for dynamic videos:

```python
camera_movements = [
    "static",         # No camera movement
    "zoom-in",        # Zoom into subject
    "zoom-out",       # Zoom out from subject
    "pan-left",       # Pan camera left
    "pan-right",      # Pan camera right
    "tilt-up",        # Tilt camera up
    "tilt-down",      # Tilt camera down
    "orbit",          # Orbit around subject
    "dolly-in",       # Move camera forward
    "dolly-out"       # Move camera backward
]

response = requests.post(
    "https://api.megsyai.com/v1/videos/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "kling-2.1-pro",
        "prompt": "Mountain landscape at golden hour",
        "camera_movement": "orbit",
        "duration": 10
    }
)
```

## Advanced Features

### Batch Generation

Generate multiple videos at once:

```python
prompts = [
    "Ocean waves at sunset",
    "Mountain landscape at dawn",
    "City skyline at night"
]

responses = []
for prompt in prompts:
    response = requests.post(
        "https://api.megsyai.com/v1/videos/generate",
        headers={"Authorization": "Bearer YOUR_API_KEY"},
        json={
            "model": "veo3-fast",
            "prompt": prompt,
            "duration": 5
        }
    )
    responses.append(response.json())
```
