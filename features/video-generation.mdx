## الميزات

### توليد من النص
أنشئ فيديو من وصف نصي:

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/videos/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "runway-gen3",
        "prompt": "موجات البحر تتكسر على الشاطئ عند الغروب، حركة بطيئة",
        "duration": 5,
        "resolution": "1080p",
        "fps": 24
    }
)

video_url = response.json()["data"]["url"]
print(f"Video URL: {video_url}")
```

### تحريك الصور
حوّل صورة ثابتة إلى فيديو متحرك:

```python
response = requests.post(
    "https://api.megsyai.com/v1/videos/animate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("image.png", "rb")},
    data={
        "motion_prompt": "الكاميرا تتحرك ببطء من اليسار إلى اليمين",
        "duration": 3
    }
)
```

### تحرير الفيديو
عدّل فيديو موجود بإضافة تأثيرات:

```python
response = requests.post(
    "https://api.megsyai.com/v1/videos/edit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"video": open("video.mp4", "rb")},
    data={
        "edit_prompt": "أضف تأثير الحركة البطيئة",
        "start_time": 2,
        "end_time": 5
    }
)
```

## المواصفات التقنية

### الدقة المتاحة

| الدقة | الأبعاد | الاستخدام |
|-------|---------|-----------|
| SD | 480p | معاينة سريعة |
| HD | 720p | جودة جيدة |
| Full HD | 1080p | جودة عالية |
| 4K | 2160p | جودة احترافية |

### المدة الزمنية

- **قصير**: 1-3 ثواني
- **متوسط**: 3-5 ثواني
- **طويل**: 5-10 ثواني
- **ممتد**: حتى 30 ثانية (للباقات المتقدمة)

### معدل الإطارات (FPS)

```python
fps_options = [
    24,  # سينمائي
    30,  # قياسي
    60   # سلس
]
```

## أمثلة عملية

```python
prompt = """

إضاءة استوديو، حركة سلسة 360 درجة، جودة 4K

response = requests.post(
    "https://api.megsyai.com/v1/videos/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "prompt": prompt,
        "duration": 5,

)
```

## معالجة متقدمة

### إضافة نص
```python
response = requests.post(
    "https://api.megsyai.com/v1/videos/add-text",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"video": open("video.mp4", "rb")},
    data={
        "text": "Megsy AI",
        "position": "center",
        "font": "Arial",
        "size": 48,
        "color": "#FFFFFF",
        "animation": "fade-in"
    }
)
```

### إضافة موسيقى
```python
response = requests.post(
    "https://api.megsyai.com/v1/videos/add-audio",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={
        "video": open("video.mp4", "rb"),
        "audio": open("music.mp3", "rb")
    },
    data={
        "volume": 0.7,
        "fade_in": 1,
        "fade_out": 1
    }
)
```

### دمج الفيديوهات
```python
response = requests.post(
    "https://api.megsyai.com/v1/videos/merge",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={
        "video1": open("video1.mp4", "rb"),
        "video2": open("video2.mp4", "rb")
    },
    data={
        "transition": "fade",
        "transition_duration": 1
    }
)
```
