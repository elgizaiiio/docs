## Create Documents

### Word Document
```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/files/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "type": "document",
        "format": "docx",
        "content": {
            "title": "Monthly Sales Report",
            "sections": [
                {
                    "heading": "Executive Summary",
                    "content": "Overview of sales performance..."
                },
                {
                    "heading": "Detailed Analysis",
                    "content": "Detailed data analysis..."
                }
            ]
        },
        "style": {
            "font": "Arial",
            "font_size": 12,
            "line_spacing": 1.5
        }
    }
)

file_url = response.json()["file_url"]
```

## File Processing

### Extract Text from PDF
```python
response = requests.post(
    "https://api.megsyai.com/v1/files/extract",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"file": open("document.pdf", "rb")},
    data={"extract_type": "text"}
)

extracted_text = response.json()["text"]
```

### Convert File Format
```python
response = requests.post(
    "https://api.megsyai.com/v1/files/convert",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"file": open("document.docx", "rb")},
    data={
        "from_format": "docx",
        "to_format": "pdf"
    }
)

converted_file_url = response.json()["file_url"]
```

### Summarize Documents
```python
response = requests.post(
    "https://api.megsyai.com/v1/files/summarize",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"file": open("long_document.pdf", "rb")},
    data={
        "summary_length": "medium",  # short, medium, long
        "language": "en"
    }
)

summary = response.json()["summary"]
```
