## إنشاء المستندات

### مستند Word
```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/files/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "type": "document",
        "format": "docx",
        "content": {
            "title": "تقرير المبيعات الشهري",
            "sections": [
                {
                    "heading": "ملخص تنفيذي",
                    "content": "نظرة عامة على أداء المبيعات..."
                },
                {
                    "heading": "التحليل التفصيلي",
                    "content": "تحليل مفصل للبيانات..."
                }
            ]
        },
        "style": {
            "font": "Arial",
            "font_size": 12,
            "line_spacing": 1.5
        }
    }
)

file_url = response.json()["file_url"]
```

## معالجة الملفات

### استخراج النص من PDF
```python
response = requests.post(
    "https://api.megsyai.com/v1/files/extract",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"file": open("document.pdf", "rb")},
    data={"extract_type": "text"}
)

extracted_text = response.json()["text"]
```

### تحويل صيغة الملف
```python
response = requests.post(
    "https://api.megsyai.com/v1/files/convert",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"file": open("document.docx", "rb")},
    data={
        "from_format": "docx",
        "to_format": "pdf"
    }
)

converted_file_url = response.json()["file_url"]
```
