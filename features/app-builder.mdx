## إنشاء موقع ويب

### من الوصف النصي
```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/apps/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "type": "website",
        "description": """
        موقع لمطعم يعرض قائمة الطعام، نظام حجز الطاولات،
        معرض صور، ونموذج تواصل. التصميم عصري وجذاب
        """,
        "features": [
            "menu_display",
            "booking_system",
            "gallery",
            "contact_form"
        ],
        "style": "modern",
        "colors": ["#FF6B6B", "#4ECDC4", "#45B7D1"]
    }
)

project_id = response.json()["project_id"]
download_url = response.json()["download_url"]
```

## API Generation

### إنشاء REST API
```python
response = requests.post(
    "https://api.megsyai.com/v1/apps/generate-api",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "type": "rest",
        "resources": [
            {
                "name": "users",
                "fields": {
                    "name": "string",
                    "email": "string",
                    "role": "enum"
                },
                "endpoints": ["list", "create", "read", "update", "delete"]
            },
            {
                "name": "posts",
                "fields": {
                    "title": "string",
                    "content": "text",
                    "author_id": "reference"
                }
            }
        ],
        "authentication": "jwt",
        "documentation": True
    }
)
```

### إنشاء GraphQL API
```python
response = requests.post(
    "https://api.megsyai.com/v1/apps/generate-api",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "type": "graphql",
        "schema": """
        type User {
            id: ID!
            name: String!
            email: String!
            posts: [Post]
        }
        
        type Post {
            id: ID!
            title: String!
            content: String!
            author: User!
        }
        """
    }
)
```
