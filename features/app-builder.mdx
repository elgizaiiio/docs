## Create a Website

### From Text Description
```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/apps/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "type": "website",
        "description": """
        A restaurant website displaying menu, table booking system,
        photo gallery, and contact form. Modern and attractive design
        """,
        "features": [
            "menu_display",
            "booking_system",
            "gallery",
            "contact_form"
        ],
        "style": "modern",
        "colors": ["#FF6B6B", "#4ECDC4", "#45B7D1"]
    }
)

project_id = response.json()["project_id"]
download_url = response.json()["download_url"]
```

## Practical Example: E-commerce Store

### Step 1: Create Project
```python
response = requests.post(
    "https://api.megsyai.com/v1/apps/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "type": "ecommerce",
        "name": "My Store",
        "description": "E-commerce store for selling clothing",
        "features": [
            "product_catalog",
            "shopping_cart",
            "checkout",
            "payment_gateway",
            "order_tracking",
            "admin_panel"
        ]
    }
)
```

### Step 2: Add Products
```python
products_api = requests.post(
    f"https://api.megsyai.com/v1/apps/{project_id}/generate-api",
    headers={"Authorization": "Bearer YOUR_API_KEY"},

        "endpoints": [
            {
                "path": "/api/products",
                "methods": ["GET", "POST", "PUT", "DELETE"],
                "authentication": True
            },
            {
                "path": "/api/orders",
                "methods": ["GET", "POST"],

        ]

```

## API Generation

### Create REST API
```python
response = requests.post(
    "https://api.megsyai.com/v1/apps/generate-api",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "type": "rest",
        "resources": [
            {
                "name": "users",
                "fields": {
                    "name": "string",
                    "email": "string",
                    "role": "enum"
                },
                "endpoints": ["list", "create", "read", "update", "delete"]
            },
            {
                "name": "posts",
                "fields": {
                    "title": "string",
                    "content": "text",
                    "author_id": "reference"
                }
            }
        ],
        "authentication": "jwt",
        "documentation": True
    }
)
```

### Create GraphQL API
```python
response = requests.post(
    "https://api.megsyai.com/v1/apps/generate-api",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "type": "graphql",
        "schema": """
        type User {
            id: ID!
            name: String!
            email: String!
            posts: [Post]
        }
        
        type Post {
            id: ID!
            title: String!
            content: String!
            author: User!
        }
        """
    }
)
```
