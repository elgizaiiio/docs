## 0v Code Generation

### What is 0v?

0v is an advanced AI coding assistant that helps you write, debug, and optimize code across multiple programming languages and frameworks.

### Generate Code

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/code/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "service": "0v",
        "prompt": "Create a React component for a user profile card with avatar, name, and bio",
        "language": "javascript",
        "framework": "react"
    }
)

code = response.json()["code"]
print(code)
```

## Supabase Integration

### Automatic Backend Generation

Megsy AI automatically generates complete backends with Supabase:

```python
response = requests.post(
    "https://api.megsyai.com/v1/projects/create",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "name": "My App",
        "description": "E-commerce platform",
        "features": [
            "user_authentication",
            "product_catalog",
            "shopping_cart",
            "payment_processing"
        ],
        "backend": {
            "provider": "supabase",
            "auto_setup": True
        }
    }
)

project_id = response.json()["project_id"]
supabase_url = response.json()["backend"]["supabase_url"]
supabase_key = response.json()["backend"]["supabase_anon_key"]
```

## GitHub Integration

### Connect Repository

```python
# Connect GitHub repository
response = requests.post(
    f"https://api.megsyai.com/v1/projects/{project_id}/github/connect",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "repository": "username/my-app",
        "branch": "main",
        "auto_commit": True,
        "commit_message_template": "Update: {description}"
    }
)
```

### Auto-Upload Files

Files are automatically uploaded to GitHub as you make changes:

```python
# Enable auto-upload
response = requests.post(
    f"https://api.megsyai.com/v1/projects/{project_id}/github/settings",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "auto_upload": True,
        "upload_on_save": True,
        "create_pull_request": False  # or True for PR workflow
    }
)
```

### Update Files

```python
# Manually trigger file update
response = requests.post(
    f"https://api.megsyai.com/v1/projects/{project_id}/github/push",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "files": [
            "src/components/Header.jsx",
            "src/pages/Home.jsx",
            "package.json"
        ],
        "commit_message": "Update components and dependencies"
    }
)
```

### Branch Management

```python
# Create new branch
response = requests.post(
    f"https://api.megsyai.com/v1/projects/{project_id}/github/branch",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "branch_name": "feature/new-ui",
        "from_branch": "main"
    }
)

# Create pull request
response = requests.post(
    f"https://api.megsyai.com/v1/projects/{project_id}/github/pull-request",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "title": "Add new UI components",
        "description": "Updated header and footer components",
        "from_branch": "feature/new-ui",
        "to_branch": "main"
    }
)
```

## Live Preview System

### Built-in Preview

View your application in real-time as you code:

```python
# Get preview URL
response = requests.get(
    f"https://api.megsyai.com/v1/projects/{project_id}/preview",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)

preview_url = response.json()["preview_url"]
# Example: https://preview-abc123.megsyai.com
```

### Hot Reload

Changes are reflected instantly in the preview:

```python
# Enable hot reload
response = requests.post(
    f"https://api.megsyai.com/v1/projects/{project_id}/preview/settings",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "hot_reload": True,
        "auto_refresh": True,
        "refresh_delay": 500  # milliseconds
    }
)
```

## Custom Domain

### Add Custom Domain

```python
# Add custom domain
response = requests.post(
    f"https://api.megsyai.com/v1/projects/{project_id}/domain",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "domain": "myapp.com",
        "subdomain": "www",  # optional
        "ssl": True
    }
)

dns_records = response.json()["dns_records"]
```

### DNS Configuration

The API returns DNS records you need to configure:

```json
{
  "dns_records": [
    {
      "type": "A",
      "name": "@",
      "value": "76.76.21.21"
    },
    {
      "type": "CNAME",
      "name": "www",
      "value": "cname.megsyai.com"
    }
  ]
}
```

### SSL Certificate

SSL certificates are automatically provisioned:

```python
# Check SSL status
response = requests.get(
    f"https://api.megsyai.com/v1/projects/{project_id}/domain/ssl",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)

ssl_status = response.json()["status"]  # "pending", "active", "failed"
```

## Complete Workflow Example

### Build a Full-Stack App

```python
# 1. Create project
project = requests.post(
    "https://api.megsyai.com/v1/projects/create",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "name": "Task Manager",
        "framework": "react",
        "backend": {"provider": "supabase"}
    }
).json()

project_id = project["project_id"]

# 2. Generate code with 0v
code = requests.post(
    "https://api.megsyai.com/v1/code/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "service": "0v",
        "prompt": "Create a task management app with add, edit, delete tasks",
        "framework": "react"
    }
).json()

# 3. Setup database
database = requests.post(
    f"https://api.megsyai.com/v1/projects/{project_id}/database/schema",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "tables": [
            {
                "name": "tasks",
                "columns": [
                    {"name": "id", "type": "uuid", "primary": True},
                    {"name": "title", "type": "text"},
                    {"name": "completed", "type": "boolean"},
                    {"name": "user_id", "type": "uuid"}
                ]
            }
        ]
    }
).json()

# 4. Connect GitHub
github = requests.post(
    f"https://api.megsyai.com/v1/projects/{project_id}/github/connect",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "repository": "username/task-manager",
        "auto_upload": True
    }
).json()

# 5. Get preview
preview = requests.get(
    f"https://api.megsyai.com/v1/projects/{project_id}/preview",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
).json()

print(f"Preview: {preview['preview_url']}")

# 6. Deploy
deployment = requests.post(
    f"https://api.megsyai.com/v1/projects/{project_id}/deploy",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={"platform": "vercel"}
).json()

print(f"Deployed: {deployment['url']}")

# 7. Add custom domain
domain = requests.post(
    f"https://api.megsyai.com/v1/projects/{project_id}/domain",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={"domain": "taskmanager.com", "ssl": True}
).json()
```
