## Chat Models

### Megsy Chat
An advanced conversational model that supports:
- Deep contextual understanding
- Answering complex questions
- Arabic and English language support
- Logical reasoning capabilities

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/chat",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "megsy-chat",
        "messages": [
            {"role": "user", "content": "What's the best way to learn programming?"}
        ]
    }
)

print(response.json()["choices"][0]["message"]["content"])
```

## Image Models

### Megsy Image
An advanced image generation model featuring:
- High quality up to 4K resolution
- Understanding of Arabic and English text
- Multiple artistic styles
- Fast generation speed

```python
response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "megsy-image",
        "prompt": "A stunning landscape of the pyramids at sunset",
        "size": "1024x1024",
        "quality": "hd"
    }
)

image_url = response.json()["data"][0]["url"]
```
