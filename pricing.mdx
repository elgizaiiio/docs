## Credit Monitoring

Track your credit usage in real-time:

```python
import requests

# Check credit balance
response = requests.get(
    "https://api.megsyai.com/v1/credits/balance",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)

balance = response.json()["credits"]
print(f"Available credits: {balance}")

# Get usage history
response = requests.get(
    "https://api.megsyai.com/v1/credits/history",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)

history = response.json()["transactions"]
```
