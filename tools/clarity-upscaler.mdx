## Usage

### Basic Upscaling

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/upscale",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("low_res.jpg", "rb")},
    data={
        "tool": "clarity-upscaler",
        "scale": 4,  # 2x, 4x, or 8x
        "enhance_details": True
    }
)

upscaled_image_url = response.json()["data"]["url"]
```

## Batch Processing

```python
# Upscale multiple images
images = ["image1.jpg", "image2.jpg", "image3.jpg"]

for img_path in images:
    response = requests.post(
        "https://api.megsyai.com/v1/images/upscale",
        headers={"Authorization": "Bearer YOUR_API_KEY"},
        files={"image": open(img_path, "rb")},
        data={"tool": "clarity-upscaler", "scale": 4}
    )
    print(f"Upscaled: {response.json()['data']['url']}")
```
