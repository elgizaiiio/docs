## Overview

AuraSR is a state-of-the-art super-resolution AI model that specializes in creating photorealistic high-resolution images from low-resolution inputs. It excels at recovering fine details and textures.

## Advanced Capabilities

- **Photorealistic Enhancement**: Natural-looking results
- **Texture Reconstruction**: Recreate realistic textures
- **Face Restoration**: Enhance facial features and details
- **Edge Preservation**: Maintain sharp, clean edges
- **Artifact Removal**: Eliminate compression artifacts
- **Color Enhancement**: Improve color vibrancy and accuracy

## Usage

### Basic Enhancement

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/enhance",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("photo.jpg", "rb")},
    data={
        "tool": "aurasr",
        "scale": 4,
        "mode": "photorealistic"
    }
)

enhanced_image_url = response.json()["data"]["url"]
```

### Portrait Enhancement

```python
response = requests.post(
    "https://api.megsyai.com/v1/images/enhance",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("portrait.jpg", "rb")},
    data={
        "tool": "aurasr",
        "scale": 4,
        "mode": "portrait",
        "enhance_face": True,
        "skin_smoothing": 0.3
    }
)
```

### Advanced Settings

```python
response = requests.post(
    "https://api.megsyai.com/v1/images/enhance",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("image.jpg", "rb")},
    data={
        "tool": "aurasr",
        "scale": 4,
        "mode": "photorealistic",
        "texture_strength": 0.8,
        "detail_level": "high",
        "color_enhancement": True,

    }
)
```
