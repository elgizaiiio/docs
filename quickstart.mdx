## Get started with Megsy AI

Follow these simple steps to start using Megsy AI's powerful features.

### Step 1: Create your account

Visit [megsyai.com](https://megsyai.com) and sign up for a free account. You can use:
- Email and password
- Google account
- GitHub account

### Step 2: Get your API key

Once logged in, navigate to your dashboard and generate your API key. Keep this key secure as you'll need it for all API requests.

### Step 3: Make your first request
