## Usage

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "seedream4",
        "prompt": "Ethereal fantasy landscape with floating islands",
        "size": "1536x1536",
        "quality": "hd"
    }
)

image_url = response.json()["data"][0]["url"]
```
