## Usage

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "flux-2-flex",
        "prompt": "Cinematic portrait with dramatic lighting",
        "size": "1024x1024",
        "guidance_scale": 7.5,
        "steps": 50,
        "seed": 12345
    }
)

image_url = response.json()["data"][0]["url"]
```

## Advanced Parameters

- **Guidance Scale**: Control prompt adherence (1-20)
- **Steps**: Number of generation steps (20-100)
- **Seed**: For reproducible results
- **Negative Prompt**: Specify unwanted elements
