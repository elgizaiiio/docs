## Usage

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "flux-pro-ultra",
        "prompt": "Ultra-realistic portrait, professional photography, 8K quality",
        "size": "2048x2048",
        "quality": "ultra",
        "guidance_scale": 8.0,
        "steps": 100
    }
)

image_url = response.json()["data"][0]["url"]
```
