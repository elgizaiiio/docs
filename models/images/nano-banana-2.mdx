## Usage

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "nano-banana-2",
        "prompt": "A cute cartoon character in a forest",
        "size": "1024x1024",
        "num_images": 1
    }
)

image_url = response.json()["data"][0]["url"]
```
