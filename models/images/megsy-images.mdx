## Usage

### Basic Generation

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "megsy-images",
        "prompt": "A beautiful sunset over the pyramids",
        "size": "1024x1024",
        "quality": "hd"
    }
)

image_url = response.json()["data"][0]["url"]
```

### Advanced Options

```python
response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "megsy-images",
        "prompt": "Professional product photography of a luxury watch",
        "negative_prompt": "blurry, low quality, distorted",
        "size": "1024x1024",
        "quality": "hd",
        "style": "realistic",
        "num_images": 4,
        "seed": 42
    }
)
```
