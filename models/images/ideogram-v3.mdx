## Usage

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "ideogram-v3",
        "prompt": "Modern tech company logo with text 'MEGSY AI'",
        "size": "1024x1024",
        "style": "design"
    }
)

image_url = response.json()["data"][0]["url"]
```
