## Usage

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/edit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("photo.jpg", "rb")},
    data={
        "model": "nano-banana-edit-pro",
        "instruction": "Remove background and enhance colors",
        "quality": "high"
    }
)

edited_image_url = response.json()["data"]["url"]
```
