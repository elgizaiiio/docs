## Usage

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/edit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("photo.jpg", "rb")},
    data={
        "model": "nano-banana-edit",
        "instruction": "Increase brightness and blur background"
    }
)

edited_image_url = response.json()["data"]["url"]
```
