## Overview

Qwen Edit is an advanced AI-powered image editing model that understands natural language instructions. Simply describe what you want to change, and the AI will make precise edits to your image.

## Capabilities

- **Object Removal**: Remove unwanted objects or people
- **Color Adjustment**: Change colors of specific elements
- **Style Transfer**: Apply different artistic styles
- **Background Change**: Replace or modify backgrounds
- **Object Addition**: Add new elements to images
- **Enhancement**: Improve lighting, contrast, and clarity

## Usage

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/edit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("original.jpg", "rb")},
    data={
        "model": "qwen-edit",
        "instruction": "Remove the person in the background and make the sky more blue"
    }
)

edited_image_url = response.json()["data"]["url"]
```
