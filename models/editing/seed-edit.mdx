## Usage

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/edit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("photo.jpg", "rb")},
    data={
        "model": "seed-edit",
        "instruction": "Transform into oil painting style with vibrant colors",
        "style_strength": 0.8
    }
)

edited_image_url = response.json()["data"]["url"]
```
