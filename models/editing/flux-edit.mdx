## Usage

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/edit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("photo.jpg", "rb")},
    data={
        "model": "flux-edit",
        "instruction": "Remove the person while preserving background details",
        "quality": "ultra",
        "preserve_details": True,
        "mask_precision": "high"
    }
)

edited_image_url = response.json()["data"]["url"]
```
