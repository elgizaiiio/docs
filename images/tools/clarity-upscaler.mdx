## How to Use

### Basic Upscaling

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/upscale",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("low_res.png", "rb")},
    data={
        "tool": "clarity-upscaler",
        "scale": 4  # 2x, 4x, or 8x
    }
)

upscaled_url = response.json()["data"]["url"]
```

### Enhancement Modes

#### Standard
- Fast processing
- Good quality improvement
- Suitable for most images

```python
{"mode": "standard"}
```

#### Enhanced
- Slower processing
- Better detail reconstruction
- Recommended for important images

```python
{"mode": "enhanced"}
```

#### Maximum
- Longest processing time
- Best possible quality
- For professional/print use

```python
{"mode": "maximum"}
```

## Advanced Parameters

### Detail Enhancement
Controls how much detail is added (0.0 - 1.0):
```python
{
    "enhance_details": True,
    "detail_strength": 0.7  # Recommended
}
```

### Noise Reduction
Remove compression artifacts and noise:
```python
{
    "denoise": True,
    "denoise_strength": 0.5  # 0.0 = none, 1.0 = maximum
}
```

### Sharpening
Add sharpness to the upscaled image:
```python
{
    "sharpen": 0.7  # 0.0 = none, 1.0 = maximum
}
```

### Color Preservation
Maintain original color accuracy:
```python
{
    "preserve_colors": True,
    "color_boost": 0.0  # Optional color enhancement
}
```

## Batch Processing

### Process Multiple Images

```python
import requests
from concurrent.futures import ThreadPoolExecutor

def upscale_image(image_path):
    with open(image_path, 'rb') as f:
        response = requests.post(
            "https://api.megsyai.com/v1/images/upscale",
            headers={"Authorization": "Bearer YOUR_API_KEY"},
            files={"image": f},
            data={"tool": "clarity-upscaler", "scale": 4}
        )
    return response.json()

# Upscale multiple images in parallel
image_paths = ["img1.png", "img2.png", "img3.png"]
with ThreadPoolExecutor(max_workers=5) as executor:
    results = list(executor.map(upscale_image, image_paths))
```

## API Reference

### Complete Example

```python
import requests

def upscale_with_clarity(image_path, scale=4, **options):
    with open(image_path, 'rb') as image_file:
        response = requests.post(
            "https://api.megsyai.com/v1/images/upscale",
            headers={
                "Authorization": "Bearer YOUR_API_KEY",
                "Content-Type": "multipart/form-data"
            },
            files={"image": image_file},
            data={
                "tool": "clarity-upscaler",
                "scale": scale,
                "mode": options.get("mode", "enhanced"),
                "enhance_details": options.get("enhance_details", True),
                "detail_strength": options.get("detail_strength", 0.7),
                "denoise": options.get("denoise", True),
                "denoise_strength": options.get("denoise_strength", 0.5),
                "sharpen": options.get("sharpen", 0.7),
                "preserve_colors": options.get("preserve_colors", True),
                "output_format": options.get("output_format", "PNG")
            }
        )
    
    return response.json()

# Usage
result = upscale_with_clarity(
    "low_res_image.png",
    scale=4,
    mode="maximum",
    enhance_details=True,
    detail_strength=0.9
)

print(f"Upscaled image: {result['data']['url']}")
print(f"Original size: {result['original_size']}")
print(f"New size: {result['new_size']}")
```

### Response Format

```json

  "new_size": "4096x4096",
  "scale_factor": 4,
  "file_size": "12.5 MB",
  "format": "PNG",
  "processing_time": 10.5
}
```
