## How to Use

### Basic Editing

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/edit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("original.png", "rb")},
    data={
        "model": "qwen-edit",
        "instruction": "Change the sky to sunset colors"
    }
)

edited_url = response.json()["data"]["url"]
```

### Advanced Editing

```python
response = requests.post(
    "https://api.megsyai.com/v1/images/edit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={
        "image": open("original.png", "rb"),
        "mask": open("mask.png", "rb")  # Optional: specify edit area
    },
    data={
        "model": "qwen-edit",
        "instruction": "Replace the background with a modern office interior",
        "strength": 0.8,
        "preserve_subject": True,
        "blend_edges": True
    }
)
```

## Parameters

### Strength
Controls how much the image changes (0.0 - 1.0):
```python
{
    "strength": 0.3,  # Subtle changes
    "strength": 0.6,  # Moderate (recommended)
    "strength": 0.9   # Strong changes
}
```

### Preserve Subject
Keep main subject unchanged:
```python
{"preserve_subject": True}
```

### Blend Edges
Smooth transitions between edited and original areas:
```python
{"blend_edges": True}
```
