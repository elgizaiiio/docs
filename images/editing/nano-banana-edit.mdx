## How to Use

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/edit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("image.png", "rb")},
    data={
        "model": "nano-banana-edit",
        "instruction": "Brighten the image"
    }
)
```
