## How to Use

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/edit",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"image": open("image.png", "rb")},
    data={
        "model": "flux-edit",
        "instruction": "Enhance and improve overall quality",
        "auto_optimize": True
    }
)
```
