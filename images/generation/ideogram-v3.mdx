## How to Use

### Basic Text Generation

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "ideogram-v3",
        "prompt": "Modern logo design with text 'MEGSY AI'",
        "text_elements": [
            {
                "text": "MEGSY AI",
                "position": "center",
                "style": "bold"
            }
        ],
        "size": "1024x1024"
    }
)

image_url = response.json()["data"][0]["url"]
```

## API Reference

### Complete Example

```python
import requests

def generate_text_image(prompt, text_elements, **options):
    payload = {
        "model": "ideogram-v3",
        "prompt": prompt,
        "text_elements": text_elements,
        "size": options.get("size", "1024x1024"),
        "quality": options.get("quality", "standard"),
        "style": options.get("style"),
        "text_effects": options.get("text_effects")
    }
    
    response = requests.post(
        "https://api.megsyai.com/v1/images/generate",
        headers={"Authorization": "Bearer YOUR_API_KEY"},
        json=payload
    )
    
    return response.json()

# Usage
result = generate_text_image(
    prompt="Modern tech logo design",
    text_elements=[{
        "text": "INNOVATE",
        "position": "center",
        "font": "sans-serif",
        "size": "xlarge",
        "weight": "bold",
        "color": "#2C3E50"
    }],
    quality="hd"
)
```

### Response Format

      {"text": "INNOVATE", "accuracy": 0.99}
