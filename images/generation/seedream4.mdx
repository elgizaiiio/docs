## How to Use

### Basic Generation

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "seedream4",
        "prompt": "Photorealistic portrait of a person in natural lighting",
        "size": "1024x1024",
        "quality": "hd"
    }
)

image_url = response.json()["data"][0]["url"]
```

### Advanced Configuration

```python
response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
```

## Advanced Parameters

### Guidance Scale
Controls how closely the image follows the prompt:
- **Low (3-5)**: More creative freedom
- **Medium (7-9)**: Balanced adherence
- **High (10-15)**: Strict prompt following

### Steps
Number of refinement iterations:
- **20-30**: Fast, good quality
- **40-50**: Balanced
- **60-80**: Maximum quality

## API Reference

### Full Request Example

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "model": "seedream4",
        "prompt": "Your detailed prompt here",
        "negative_prompt": "blurry, low quality",
        "size": "1024x1024",
        "quality": "hd",
        "guidance_scale": 7.5,
        "steps": 50,
        "seed": None,  # Random seed
        "num_images": 1
    }
)

result = response.json()
print(f"Image URL: {result['data'][0]['url']}")
print(f"Generation time: {result['generation_time']}s")
```
