## How to Use

### Basic Usage

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "nano-banana-pro",
        "prompt": "Professional digital artwork of a fantasy landscape",
        "size": "1024x1024",
        "quality": "hd"
    }
)

image_url = response.json()["data"][0]["url"]
```

## API Reference

### Complete Example

```python
import requests

def generate_image(prompt, **kwargs):
    response = requests.post(
        "https://api.megsyai.com/v1/images/generate",
        headers={"Authorization": "Bearer YOUR_API_KEY"},
        json={
            "model": "nano-banana-pro",
            "prompt": prompt,
            "negative_prompt": kwargs.get("negative_prompt", ""),
            "size": kwargs.get("size", "1024x1024"),
            "quality": kwargs.get("quality", "standard"),
            "style": kwargs.get("style", "digital-art"),
            "creativity": kwargs.get("creativity", 0.5),
            "detail_level": kwargs.get("detail_level", "medium"),
            "num_images": kwargs.get("num_images", 1)
        }
    )
    return response.json()

# Usage
result = generate_image(
    "Fantasy character design",
    quality="hd",
    style="fantasy",
    creativity=0.7,
    detail_level="high"
)
```
