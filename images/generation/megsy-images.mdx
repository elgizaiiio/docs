## How to Use

### Basic Generation

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "megsy-images",
        "prompt": "A beautiful sunset over the ocean with vibrant colors",
        "size": "1024x1024",
        "quality": "hd"
    }
)

image_url = response.json()["data"][0]["url"]
print(f"Generated image: {image_url}")
```

### Advanced Options

```python
response = requests.post(
```

## API Reference

### Parameters

- `model` (required): "megsy-images"
- `prompt` (required): Text description of the image
- `negative_prompt` (optional): What to avoid in the image
- `size` (optional): Image dimensions (default: "1024x1024")
- `quality` (optional): "standard" or "hd" (default: "standard")
- `style` (optional): Style preset
- `num_images` (optional): Number of images to generate (1-4)

### Response

```json
{
  "data": [
    {
      "url": "https://cdn.megsyai.com/images/...",
      "revised_prompt": "Enhanced version of your prompt"
    }
  ],
  "model": "megsy-images",
  "created": 1234567890
}
```
