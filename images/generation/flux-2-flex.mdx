## How to Use

### Basic Generation

```python
import requests

response = requests.post(
    "https://api.megsyai.com/v1/images/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "flux-2-flex",
        "prompt": "A serene Japanese garden with cherry blossoms",
        "size": "1024x1024"
    }
)

image_url = response.json()["data"][0]["url"]
```

## Style Control

### Style Strength
Controls how strongly the style is applied (0.0 - 1.0):
```python
{
    "style_strength": 0.3,  # Subtle style
    "style_strength": 0.6,  # Balanced (recommended)
    "style_strength": 0.9   # Strong style
}
```

### Style Mixing
Combine multiple styles:
```python
{
    "styles": [
        {"name": "photographic", "weight": 0.6},
        {"name": "cinematic", "weight": 0.4}
    ]
}
```

## API Reference

### Complete Example

```python
import requests

def generate_flex_image(prompt, **options):
    payload = {
        "model": "flux-2-flex",
        "prompt": prompt,
        "negative_prompt": options.get("negative_prompt", ""),
        "size": options.get("size", "1024x1024"),
        "flex_mode": options.get("flex_mode", "adaptive"),
        "style_strength": options.get("style_strength", 0.6),
        "composition_guide": options.get("composition_guide"),
        "color_palette": options.get("color_palette"),
        "lighting_mode": options.get("lighting_mode"),
        "depth_of_field": options.get("depth_of_field")
    }
    
    response = requests.post(
        "https://api.megsyai.com/v1/images/generate",
        headers={"Authorization": "Bearer YOUR_API_KEY"},
        json=payload
    )
    
    return response.json()

# Usage
result = generate_flex_image(
    "Sunset over ocean",
    flex_mode="adaptive",
    composition_guide="rule-of-thirds",
    color_palette="warm",
    lighting_mode="golden-hour"
)
```

### Response Format

```json
{
  "data": [{
    "url": "https://cdn.megsyai.com/images/...",
    "flex_settings": {
      "mode": "adaptive",
      "style_strength": 0.6,
      "composition": "rule-of-thirds"
    }
  }],
  "model": "flux-2-flex",
  "generation_time": 8.5
}
```
